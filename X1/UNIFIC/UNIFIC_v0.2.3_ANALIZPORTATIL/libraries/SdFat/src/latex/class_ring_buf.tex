\doxysubsection{Descripción detallada}
\subsubsection*{template$<$class F, size\+\_\+t Size$>$\newline
class Ring\+Buf$<$ F, Size $>$}
Ring buffer for data loggers. 

This ring buffer may be used in ISRs. \mbox{\hyperlink{class_ring_buf_af24035c0a756431c8c73b624fb6df303}{bytes\+Free\+Isr()}}, \mbox{\hyperlink{class_ring_buf_a759455d54cc9fc3903247d00c28897f5}{bytes\+Used\+Isr()}}, memcopy\+In(), and memcopy\+Out() are ISR callable. For ISR use call memcopy\+In() in the ISR and use \mbox{\hyperlink{class_ring_buf_a3ab8d4f4b7bc833018fe4db182660051}{write\+Out()}} in non-\/interrupt code to write data to a file. \mbox{\hyperlink{class_ring_buf_a60f4cd44371930a700dbb82aba025557}{read\+In()}} and memcopy\+Out can be use in a similar way to provide file data to an ISR.

Print into a \mbox{\hyperlink{class_ring_buf}{Ring\+Buf}} in an ISR should also work but has not been verified.
